# E-commerce project

### Features 

- Responsive UI

- Log in & Register

- Categories Listing

- Products Listing

- Filtering of products by categories

- Cart 

### Local set up

- Clone the repo

- Install dependencies using `npm install`

- Run the project using `npm start`

- Open `localhost:3000` for UI, `localhost:5000` for api server
